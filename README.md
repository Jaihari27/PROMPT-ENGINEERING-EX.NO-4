Explanation (Use Case from Unit 5)
Use Case: AI-based Smart Manufacturing Automation System
The manufacturing industry is increasingly adopting Artificial Intelligence (AI), Internet of Things (IoT), and embedded systems to automate operations and reduce manual monitoring.
In this system:
• IoT sensors collect data from machines.
• Embedded controllers monitor machine performance.
• AI algorithms analyze data to detect faults and predict maintenance.
• Cloud platforms allow remote monitoring and control.
This system helps industries:
• Reduce downtime
• Improve productivity
• Optimize energy consumption
• Enable predictive maintenance

Procedure
Define the scenario and use case.
Identify the required prompt engineering patterns.
Design prompts using different prompt types.
Provide prompts to different AI tools (example: ChatGPT, Gemini, Copilot).
Collect the generated responses.
Evaluate the responses using a Rubrics evaluation method.
Compare the results and analyze the effectiveness of prompts.

Prompt Design Using Different Prompt Types
1. Comparative Analysis Prompt
Definition
A prompt that asks AI to compare two or more technologies, highlighting their similarities and differences.
Prompt
“Compare and contrast AI-based predictive maintenance and traditional scheduled maintenance in manufacturing industries. Explain their advantages, disadvantages, cost efficiency, and impact on production downtime.”
Expected Output
The AI should explain:
• Predictive maintenance uses AI and sensor data
• Scheduled maintenance follows fixed time intervals
• Predictive maintenance reduces downtime
• Traditional maintenance may lead to unnecessary servicing

2. Experiential Perspective Prompt
Definition
A prompt that asks AI to explain a concept from the viewpoint of a specific person or profession.
Prompt
“As a manufacturing plant engineer, explain how AI-based predictive maintenance helps in reducing machine failures and improving production efficiency in a factory.”
Expected Output
The AI should describe:
• How engineers monitor machines using AI dashboards
• How early fault detection prevents equipment failure
• How production efficiency improves

4. Everyday Functioning Prompt
Definition
Prompts that explain how a technology works in daily life or real-world operations.
Prompt
“Explain how IoT sensors and AI systems work together in a smart factory to monitor machines and prevent unexpected failures.”
Expected Output
The AI should explain:
• Sensors collect vibration, temperature, and performance data
• AI analyzes the data
• Alerts are generated before failures occur
• Engineers take preventive actions

4. Universal Prompt Structures
Definition
Universal prompt structures are standard prompt templates used to get clear and structured responses.
Example Prompts
Prompt 1
“Explain AI-based predictive maintenance in simple terms with an example.”
Prompt 2
“Explain the process of smart manufacturing automation using IoT and AI step by step.”
Expected Output
The AI should provide:
• Simple explanation
• Step-by-step working
• Real-world examples

5. Prompt Size Limitations
Definition
AI models have token or input size limits, so prompts must sometimes be broken into smaller parts.
Example
Large prompt:
“Explain 50 applications of AI in manufacturing.”
Improved prompt:
“Explain 10 applications of AI in manufacturing industries.”
Next prompt:
“Give the next 10 AI applications in manufacturing.”
Purpose
• Improves response clarity
• Prevents AI overload
• Generates better structured answers

Result
The experiment successfully generated and tested different prompt engineering techniques including Comparative Analysis Prompt, Experiential Perspective Prompt, Everyday Functioning Prompt, Universal Prompt Structures, and Prompt Size Limitations for the selected use case of AI-based Smart Manufacturing Automation.
The prompts were provided to different AI tools and the responses were evaluated using a Rubrics evaluation method based on criteria such as accuracy, clarity, relevance, and completeness.

Thus, the experiment demonstrates that prompt engineering plays an important role in improving the effectiveness and quality of AI-generated responses in real-world applications such as smart manufacturing automation.
